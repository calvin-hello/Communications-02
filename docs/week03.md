## Definition of terms

* _**Persona**_: is a description of a possible user of a product or service. Personas are created to build
empathy, identify user characteristics, behaviours, and goals.

* _**User Stories**_: identify technical features of a product or service we need to build. They are a series of
steps that identify things we need to build and make sure that they are implemented on the final project.

* _**User Scenarios**_: are technical descriptions of steps required to complete a User Story.
